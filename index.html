Pulse Wave Database

A database of simulated pulse waves

The Pulse Wave Database project.

The aim of this project is to develop a database of simulated pulse waves for in silico testing of pulse wave analysis algorithms. A database of simulated pulse waves representative of a sample of healthy adults is being created. It contains several types of pulse waves at common measurement sites for virtual subjects aged 25 - 75 years old. The database will be made publicly available, and is complimented by publicly available resources including pulse wave analysis algorithms.
