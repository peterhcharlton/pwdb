Preliminary Pulse Wave Database

A database of simulated pulse waves representing 25- to 75-year olds with varying cardiovascular properties.

Datasets Homepage

Overview

The Preliminary Pulse Wave Database is a set of simulated arterial pulse waves for 258 virtual subjets aged from 25 to 75 years old (in 10 year increments). The database contains a baseline simulation for each of the six age groups, which was created using cardiovascular properties (such as heart rate and arterial stiffness) which are representative of healthy subjects at each age group. It also contains a further 42 virtual subjects at each age group, in which one of the cardiovascuar properties was varied from its baseline value whilst all others were held constant at their baseline values. This allows for preliminary analyses of the performance of pulse wave analysis algorithms.

Details

The dataset contains arterial flow velocity (U), luminal area (A), pressure (P) and photoplethysmogram (PPG) pulse waves at a range of measurement sites, including: (i) aorta (ascending and descending); (ii) carotid artery; (iii) brachial artery; (iv) radial artery; (v) finger; and (vi) femoral artery.

An overview of the dataset, inclduing details on its design and validation, is provided in the talk accompanying this publication.

Limitations

The utility of this dataset is restricted because the virtual subjects cannot be considered to be representative of a sample of healthy adults, as cardiovascular properties were only varied indpendently, whereas in reality an individual's cardiovascular properties are all slightly different from the norm for their age.

Publication

The dataset was first reported at the 2018 World Congress of Biomechanics. Further detail was subsequently provided in this publication, which contains additional information on how the dataset was designed and validated.

Citation

When using this dataset please cite this publication:

Charlton P.H. et al. Modelling arterial pulse wave propagation during healthy ageing, In World Congress of Biomechanics 2018, Dublin, Ireland, 2018.
