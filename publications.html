Project Publications

The following publications describe the design and verification of the full database of simulated pulse waves:

  • Modelling arterial pulse waves in healthy ageing: a database for in silico evaluation of haemodynamics and cardiovascular indices (In Preparation): This article will describe the pulse wave database (v.1), and illustrate its utility through a range of case studies.

In addition, the following publications describe the preliminary version of the database, which is available now:

  • A Database for the Development of Pulse Wave Analysis Algorithms: This presentation provides a helpful overview of the preliminary database. Many of the design principles used for this preliminary database remained the same when constructing the full database.
  • A Database for the Development of Pulse Wave Analysis Algorithms: This abstract was presented at the 2018 World Congress on Biomechanics.

Related Publications

In addition, the following publications have made use of the pulse wave databases created in this project:

  • A numerical assessment of methods to estimate aortic stiffness from arterial pulse waves: This poster reports results of using the preliminary pulse wave database to assess the performance of a range of methods for estimating aortic stiffness from pulse waves. DOI: 10.5281/zenodo.1311197.
